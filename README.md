# Контроль качества звонков

ИИ-супервайзер прослушивает записи звонков менеджеров и оценивает работу
по чеклисту: ставит удержания, выделяет сильные стороны, зоны роста и
конкретные рекомендации с примерами фраз.

## Возможности

- Загрузка нескольких mp3/wav/m4a за раз с привязкой к менеджеру
- Автоматическая расшифровка и анализ через Google Gemini 2.5 Flash
- Дэшборд: KPI, графики динамики, рейтинг менеджеров
- Детальный просмотр каждого звонка с цитатами и рекомендациями
- Экспорт в Excel

## Стек

- Python + Streamlit
- Google Gemini API (бесплатный тариф)
- SQLite (локальная база)

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

После запуска откроется http://localhost:8501

API-ключ можно ввести в боковой панели или через `.streamlit/secrets.toml`:

```toml
GEMINI_API_KEY = "your_key_here"
```
