## Технический стек

| Компонент | Что | Где |
|---|---|---|
| UI + хостинг | Streamlit multi-page | Streamlit Community Cloud (бесплатно) |
| База данных | SQLite (файл `kk_calls.db`) | Локально на хостинге |
| ИИ | Google Gemini 2.0/2.5 Flash | API через aistudio.google.com |
| Транскрипция | Gemini (встроенная) | Аудио → текст с диаризацией |
| Графики | Plotly | В Streamlit |
| Валидация JSON | Pydantic + кастомный парсер | core/json_validator.py |

**API-ключ Gemini** — в Streamlit Secrets (`GEMINI_API_KEY`) или вводится вручную в боковой панели.

## AI Pipeline (как работает анализ)

```
Загрузка аудиофайла (MP3/WAV/M4A/OGG/FLAC)
    ↓
Gemini: транскрипция с диаризацией (Менеджер / Клиент + таймкоды)
    ↓
Парсинг транскрипта → сегменты → сохранение в transcripts + transcript_segments
    ↓
Построение промпта (чеклист + веса этапов из БД + правила оценки)
    ↓
Gemini: полный анализ → строго валидный JSON (21 поле + массивы)
    ↓
Валидация + авторемонт JSON (json_validator.py)
    ↓
Пересчёт QA Score по весам из БД (score_calculator.py)
    ↓
Сохранение: qa_analyses, sales_stage_scores, detected_errors,
            sales_tools, objections, emotional_timeline,
            call_timeline_events, phrase_library, rop_recommendations
    ↓
Обновление дашбордов
```

**Цепочка моделей с фолбэком:** gemini-2.5-flash → gemini-2.0-flash → gemini-1.5-flash

## База данных (SQLite, файл kk_calls.db)

Ключевые таблицы:
- `teams` + `users` — команды и менеджеры
- `calls` — все звонки
- `transcripts` + `transcript_segments` — транскрипты с таймкодами
- `checklist_rules` — чеклист (редактируется в UI)
- `stage_weights` — веса этапов по типу звонка
- `qa_analyses` — итоги анализа (10+ числовых метрик)
- `sales_stage_scores` — баллы по каждому этапу
- `detected_errors` — ошибки со статусом (AI / ручное подтверждение)
- `sales_tools` — инструменты продаж (использованы / пропущены)
- `objections` — явные + скрытые возражения
- `emotional_timeline` — эмоциональное состояние по ходу звонка
- `call_timeline_events` — лента событий звонка
- `phrase_library` — лучшие и худшие фразы
- `rop_recommendations` — рекомендации для РОПа
- `training_recommendations` — AI-тренинги

## Известные ограничения

1. **SQLite на Streamlit Cloud** — данные на эфемерном диске, теряются при редеплое. Решение: регулярный экспорт через страницу «Настройки → БД», или миграция на Supabase.
2. **Free tier Gemini** — ~20 запросов/день. При исчерпании — система переключается на следующую модель в цепочке. Мониторинг: https://ai.dev/rate-limit
3. **Аудиофайлы не хранятся** — при анализе аудио отправляется в Gemini, но не сохраняется на сервере (нет хранилища). Плеер работает только если файл доступен по `audio_path`.
